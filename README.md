# 👩‍🏫 极简班级计分板 (Minimalist Class Scoreboard)

> 一款基于 Python Flask 和 SQLite 的轻量级、本地化课堂计分系统。专为真实教学场景设计，支持多端局域网同步、拼音快捷定位、以及自然语言 AI 记分辅助。

## ✨ 核心特性

- **📦 极简零配置部署**：前端、后端、数据库交互全部封装在一个 `app.py` 文件中，开箱即用。
- **🚀 极速课堂打分**：
  - **拼音首字母定位**：在键盘上敲击学生姓名的首字母（如 `s`），页面瞬间平滑滚动并高亮闪烁目标学生。
  - **批量操作**：提供大尺寸复选框，支持一次性勾选多名学生进行批量加减分。
  - **立体数据面板**：直观展示学生“总分”，并分类汇总“累计加分 (+)”与“累计扣分 (-)”。
- **🤖 智能 AI 助手**：内置正则表达式语义解析引擎，支持自然语言打分（如输入 *“给李四上课走神扣1分”*）。支持多音字识别与拼音模糊匹配防错。
- **📱 跨设备实时同步**：电脑作为本地服务器，手机扫码/输入 IP 即可作为“打分遥控器”。手机端切回浏览器时自动拉取最新数据，告别数据不同步。
- **🛡️ 一定的数据安全**：
  - **防呆导入**：智能识别 Excel 复制的空格与 Tab，遇同号不同班自动加前缀防覆盖。
  - **时光倒流**：每次关闭系统（`Ctrl+C`）自动在 `backups` 文件夹生成带时间戳的数据库备份，系统自动清理，仅保留最近 10 份。

## 🛠️ 技术栈

- **后端**: Python 3, Flask, SQLite3, pypinyin (用于汉字转拼音及多音字纠错)
- **前端**: HTML5, 原生 CSS3 (CSS Grid 响应式布局), Vanilla JavaScript
- **存储**: 本地 `.db` 单文件数据库

## 🚀 快速开始

### 1. 环境准备
确保你的电脑上已安装 Python 3。打开终端（命令提示符），安装必要的依赖库：
```bash
pip install flask pypinyin
```

### 2. 运行系统
在 `app.py` 所在的目录下，运行以下命令：
```bash
python app.py
```
系统启动后，在浏览器中访问 `http://127.0.0.1:5000` 即可进入主界面。

### 3. 局域网手机访问（遥控器模式）
1. 确保手机与运行该系统的电脑连接在**同一个 Wi-Fi** 下。
2. 查看电脑的局域网 IPv4 地址（Windows 下运行 `ipconfig`，Mac 下运行 `ifconfig`）。
3. 在手机浏览器中输入 `http://<你的电脑IP>:5000` 即可随时随地进行课堂打分。

## 💡 使用指南与高阶技巧

* **初始化班级**：点击底部的“⚙️ 展开高级设置”，在【批量导入学生名单】中，从 Excel 直接复制 `学号`、`姓名`、`班级` 三列粘贴导入。
* **历史分数继承**：如果期中已经有了一批分数，可以在【批量导入已有分数】中直接粘贴 `学号` 和 `分数`，系统会自动建账。
* **数据撤销与恢复**：
  * **单条撤销**：鼠标悬停在学生卡片上，或点击进入该生全部明细，即可撤销任意一笔误操作。
  * **全盘恢复**：若发生严重误操作，关闭系统后，进入 `backups` 文件夹，将对应的历史备份文件复制到主目录并重命名为 `score_system.db` 即可瞬间恢复。
* **多音字纠错**：系统已内置如“阚(kàn)”、“单(shàn)”、“解(xiè)”等常见多音字姓氏字典，确保拼音排序与检索绝对精准。
